# Aditya's Developer Portfolio

A premium, modern, and interactive portfolio website showcasing Aditya's skills, projects, and experiences as a Python Developer.

## 🚀 Features

- **Dynamic Typing Animation**: Smooth interactive text highlighting key specializations.
- **Scroll Reveal Animations**: Elegant micro-interactions and transitions as the user scrolls.
- **Fully Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
- **Tech Stack Grid**: Interactive cards displaying core skills, including FastAPI, Django, Docker, AWS, and Git.
- **Featured Projects**: Highlights of key projects with external links and code repository tags.
- **Clean Structure**: Single-page modern design utilizing HSL colors, dark mode aesthetic, and CSS variables.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom variables, Keyframes), JavaScript (ES6+)
- **Icons**: Font Awesome 6
- **Typography**: Google Fonts (Inter, Fira Code)
